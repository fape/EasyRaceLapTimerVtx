# EasyRaceLapTimerVtx

use framebuffer to display [EasyRaceLapTimer](https://github.com/polyvision/EasyRaceLapTimer) monitor view

Raspberry Pi
- force pal mode
/boot/config.txt
```
sdtv_mode=2
hdmi_ignore_hotplug=1
overscan_scale=1
overscan_left=25
overscan_right=16
overscan_top=0
overscan_bottom=0
```
